## **주요 구성 요소 요약**

* **세션 사이드바 (좌측):**
  * `Conversations` 타이틀과 세션 목록 (Meeting Notes, API Plan 등).
  * 목록 하단에 + New Chat` 버튼 배치: 클릭시 오른쪽 채팅창에서 새로운 대화 시작.
  * [✎] 편집 버튼과 [🗑] 삭제 버튼으로 각 세션 관리 가능.
* **채팅창 (우측):**
  * AI와 User의 대화 내역 및 타임스탬프 표시.
* **하단 컨트롤 바:**
  * 연결 상태 표시 (`Connected`).
  * Drag -> 해당 버튼을 누르고 움직이면 채팅창 위치를 드래그가능한 버튼
  * ChatHistory -> 채팅히스토리 Toggle 버튼
  * SessionList -> 세션 목록 Toggle 버튼
  * Send -> 메시지 입력 후 전송 버튼
  * Stop -> 메시지 입력 후 AI processing중일때 Message입력창은 비활성화되고 Send 버튼은 Stop버튼으로 바뀌어, AI processing을 중단할 수 있는 버튼

이떄 위 레이아웃버튼들은 추후 직관적인 UI 이미지들로 대체될 수 있습니다.

## 채팅창, 세션사이드바는 모두 Toggle여부와 관련없이, 작동해야함

시나리오

1. 세션사이드바는 토글, 채팅창은 비토글인상태에서, 세션사이드바에서 New Chat 버튼을 누르면, 채팅창은 토글과 상관없이, 새로운 대화가 시작되어야함.
2. 세션사이드바는 토글, 채팅창은 비토글인상태에서, 세션사이드바에서 기존 대화 클릭하면, 채팅창은 토글과 상관없이, 해당 대화로 이동해야함.
3. 세션사이드바는 비토글, 채팅창은 토글인상태에서, 새 메시지를 입력하면 채팅창에는 새 메시지가 추가되어야하고, 세션사이드바의 업데이트날짜가 해당 시간으로 업데이트 되야함.
4. 세션사이드바는 비토글, 채팅창은 비토글인상태에서, 새 메시지를 입력하면 채팅창에는 새 메시지가 추가되어야하고, 세션사이드바의 업데이트날짜가 해당 시간으로 업데이트되어야함.
